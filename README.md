# FRONTEND BOILERPLATE

## Install
```sh
$ git clone 
$ rm -rf .git
$ git init
$ yarn install
```

## Develop
```sh
yarn start       # for developing and   open  localhost:3000
yarn build       # for build. creating folder "build"
```
